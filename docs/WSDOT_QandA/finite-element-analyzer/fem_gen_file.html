Finite Element Analysis - File Generation

The best way to create a F3D file is to use the FEA report file from a bridge analysis. Best to use a BDF file for a small bridge so the resulting FEA file is not to cumbersome to edit. Before doing a run, either LFR or LRFR, set the AnalysisControls switch “RptFEA” to “Y”
